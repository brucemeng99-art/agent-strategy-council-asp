# Agent Strategy Council ASP

A public, sanitized multi-agent strategy council demo for OKX.AI Genesis Hackathon.

It turns one business question into a structured boardroom-style decision:

1. Role-based agent perspectives
2. Cross-review and debate
3. Chair synthesis
4. Executable action plan
5. Per-session pricing concept

This repository intentionally contains no private company context, no local bridge paths, no internal agent names, and no API keys.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Safety boundary

The hackathon MVP is a public demonstration layer. It does not connect to private local automation, internal file buses, company data, private networks, or execution agents.
