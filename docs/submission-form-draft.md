# OKX.AI Genesis Hackathon — Submission Draft

## Project name

Agent Strategy Council

## Subtitle

Powered by AI Organization Theory

## One-line description

An approval-gated AI strategy council that turns scattered AI tools into an organized digital workforce with roles, decision rounds, execution tasks, and organizational memory.

## Project summary

Agent Strategy Council helps business owners organize AI work like an operating system instead of using isolated chatbots. A user submits a business question, multiple AI roles debate it, the chairman can open follow-up rounds, one chair decision is produced, and execution tasks are unlocked only after approval. The meeting is preserved as organizational memory.

## Problem

AI adoption inside companies is fragmented. Teams use multiple AI tools, but work is not organized:

- no clear role division;
- no decision protocol;
- no approval gate;
- no execution tracking;
- no reusable memory.

This creates confusion, duplicated work, and unsafe autonomous execution.

## Solution

Agent Strategy Council adds an organizational layer above AI tools:

1. Board layer: human approval.
2. Council layer: multi-role debate.
3. Source and memory layer: evidence and prior decisions.
4. Execution layer: approved tasks for coding agents, research agents, and workflow tools.
